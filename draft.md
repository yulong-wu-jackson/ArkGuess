明日方舟猜猜我是谁 游戏

tech stack:
TypeScript + Vite + Tailwind CSS + shadcn/ui

我们想要制作一个“猜猜我是谁 游戏”，这个游戏会通过GitHub action host 在 GitHub上

我们需要一个picture图片库，图片库里有不同的folder，每个folder对应一个主题，比如图片库里第一个主题是明日方舟，里面会有 带有明日方舟干员名字的图片

游戏规则 （user flow）
1.随机模式
用户选择图片库里的主题（如明日方舟），并选择几x几的格子 并随机分配主题库里的图片到格里，每个格子里有一位干员的图片，并在下面标注干员名字

选择后开始游戏，这个游戏会有两方，（比如红蓝两方 红色是x标记，蓝色是o标记），然后玩家可以选择x/o 工具后点击干员，便可以标记干员，选择x/o工具后点击已经被标记的干员可以取消对应的那个标记
然后还有个结束游戏按钮，点击后向用户确认，确认后返回选择主题的初始页面

2.自选干员模式
用户选择几x几的格子，然后向用户展示对应主题folder里的所有干员然后可以进行多选选择，然后后面会和之前随机模式一样
